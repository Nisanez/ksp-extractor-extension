# Icons

These are programmatically generated placeholder icons (blue square with a
dark frame). Replace with the final brand icon before publishing to the
Chrome Web Store.

Required sizes (already in place):
- `icon16.png` — toolbar
- `icon48.png` — extension management page
- `icon128.png` — Web Store listing + install dialog

You can regenerate them by running the snippet in this repo's PR history,
or drop in your own PNGs with the same filenames.
